# 강의들

## indexedDB
사용자의 브라우저에 데이터를 영구적으로 저장할 수 있는 방법<br>
ex) 로그인 안하고 장바구니<br>
same-origin policy<br>
50mb 정도의 용량, 대용량 데이터 이슈<br>

## Storage
local, session으로 나누어짐<br>
local: 기기와 플랫폼이 허락하는 한 데이터 저장<br>
ex)사용자 정보를 기록할 때<br>
session: 브라우저를 닫으면 데이터가 삭제<br>
5~10mb 사용가능<br>
